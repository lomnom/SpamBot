# SpamBot, the speedy offhand spammer!
## Usage
### OSX
1. `spam.bash`
## Installation
### OSX
```bash
git clone https://github.com/lomnom/SpamBot
cd SpamBot
chmod +x spam.bash
cp ./* /usr/bin/
```
## What it does
It takes control of your keyboard, and spams the message that you want to spam by typing it and pressing enter
